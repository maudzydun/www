Формы

  • атрибут екшн для формы - тот файл где будет происходить обработка всей формы
  • атрибут autocomplete="off" по умолчанию этот параметр включен, это подсказка по клику на поле (может выпадать имей или телефон из кеша), можно отключить с помощью офф
  • метод - тип отправки данных в этот файл
    • get передается через юрл адрес
    • post обычная передача данных без привязки к юрл
  • атрибуты для инпута
    • type= "text" - обычное текстовое поле,
        "email" - поле для ввода имейла,
        "phone" - телефон,
        "date" - можем выбирать просто дату
        "time" - можем установить время
        "datetime" - позволяет выбрать и дату и время
        "number" - вводить исключительно числа
        "color" - поле для выбора цвета
        "file" - можно отправить файл на сервер, если выбран файл то в таком случае обязательно в атрибутах form указываем enctype="multipart/form-data",
        "hiden" - скрытое поле которое не видно на самой странице, с обязательным параметром value - это значение будет передаваться вместе с остальными значениями
        "password" - пользователь при вводе будет видеть кружки
        "range" - поле с выбором диапазона
        "reset" - создает кнопку сброса данных в полях без перезагрузки страницы. Так же можно использовать в теге батон. В теге инпут название кнопки прописываем в поле value
    • name= указываем как будет называться наше поле для ввода, можно указать что хотите получить - например имя
    • value= значение которое будет находится по умолчанию в этом поле, лучше оставлять пустым и пользоваться плейсхолдером
    • placeholder= полупрозрачная надпись на фоне
  • атрибут for для лейбл можно использовать для любого тега , он по атрибуту айди находит обьект и становится от него зависимым
  • для кнопки с тегом button есть атрибут тайп= так вот если тайп ее будет баттон - ничего не произойдет , просто будет нажиматься кнопка , если будет тайп сабмит - данные отправятся на сервер
  • атрибут disabled - делает поле не активным и его нельзя заполнить или изменить
