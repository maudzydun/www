Кнопки Бутстрап

Чтобы кнопка унаследовала основные свойства нужно прописать клас btn

Цвета

btn-outline-primary btn-outline-info btn-outline-secondary btn-outline-success btn-outline-danger btn-outline-warning

Характеристики для кнопок

для ссылки role="button" aria-disabled="true" tabindex='-1' и в класе тоже прописать disabled для ссылки можно просто в характеристиках и в классе просто disabled

Групировка кнопок

родительский класс указываем как btn-group

К группе кнопок тоже можно применять различные свойства:
btn-group-lg - группа больших кнопок
btn-group-sm - группа маленьких кнопок
btn-group-vertical - вертикальная группа кнопок

для родительского класса указываем btn-toolbar

Кнопки групируются в один блок при этом видно небольшое разделение между ними
btn-group-vertical - каждый батон груп будет распологаться горизонтально , а второй ряд кнопок будет под ним и третий тоже будет ниже соответственно при этом наши кнопки будут растянуты на 100% ширины контейнера
если нужно еще создать дополнительные отступы между группами используем клас мерджин для груп
