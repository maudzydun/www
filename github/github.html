Установка и настройка

Для установки git его нужно скачать и потом работать в обычном терминале cmd или в терминале bash. Чтобы проверить версию и работу git нужно прописать в терминале git --version . Дальше нам необходимо создать репозиторий , для этого существует такая команда как git init . Важно ! Проэкт загружает в облако все файлы и папки внутри папки в которой открыт терминал и создает репозиторий. После выполнения этой команды создается внутри проэкта скрытая папка точкагит . Так же необходимо выполнить настройку аккаунта. Прописываем git config --global user.name 'maudzydun' . И Прописываем git config --global user.email 'semenzelenskij@gmail.com'

Добавление данных


добавление файлов в проэкт происходит с помощью команды git add (и здесь указываем что мы добавлем):

  • . - добавляем все файлы в репозиторий
  • *.txt - добавление всех файлов с расширением например txt
  • myimg/ - добавление папки например myimg
  • myimg/*.html - добавление всех штмл файлов из папки myimg
  • myimg/**/*.html - добавление всех штмл файлов из папки myimg и всех ее подпапок
  • index.html - добавляет только один наш файл
