Подключение JavaScript

  1. Первый способ это прописать тег script> И внутри его прописывать весь джава скрипт код
  2. Создать отдельный файл в котором прописывать весь джава скрипт и потом это файл подулючать с наш документ. Для этого создаем тег скрипт и не внутри тега , а в характеристиках к открывающему тегу прописываем src="путь к нашему обьекту". Так как обьект может находится не только у нас на пк но и на удаленном сервере - мы можем прописать путь через http://путь к файлу .
  3. Так как js прописывать нужно в самом конце странички то и загружается он самым последним на страничке. Но если нам нужно чтобы js загружался сразу мы можем прописать атрибут к нашему тегу script который называется async который позволяет загружать наш скрипт асинхронно со всей страничкой html. Этим атрибутом не рекомендуется злоупотреблять чтобы не замедлить загрузку всей странички
  4. Одновременно мы можем подключать сразу несколько script файлов

Переменные в JavaScript

Чтобы создать переменную нужно прописать ключевое слово:
  1. var - простейшая переменная
  2. let - тоже простейшая переменная, но отличается областью видимости что мы сможем заметить в циклах или в функциях
  3. const - переменная константа которую нельзя потом переопределить
